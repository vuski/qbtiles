# Example: Web Client

Browser-based QBTiles index reader. Open the HTML file directly or serve it locally.

Features:

- **Built-in demo**: Constructs a binary index in JavaScript and deserializes it — no Python needed
- **File loader**: Load an `index.gz` file generated by the Python builder
- **Tile lookup**: Query z/x/y coordinates to get offset/length for HTTP Range Requests

[Open 03_web_client.html](https://github.com/vuski/qbtiles/blob/main/examples/03_web_client.html)
